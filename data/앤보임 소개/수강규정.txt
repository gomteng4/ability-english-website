=== 수강규정 (앤보임 소개) ===

회원가입	
1인 1 ID 사용(한 ID로 두 명 수강 불가)
수강권 종류	
1회 25분 / 4회 20분·25분 / 8회 20분·25분 / 12회 20분·25분 / 20회 20분·25분 / 24회 20분·25분 /
36회 20분·25분
수강등록	
최대 일주일 이내 시작 수강 등록 가능
당일 수강 등록은 시작 2시간 전까지 등록가능
수강 요일 타입:
- 주1회 : 월/화/수/목/금/토
- 주2회 : 화,목/ 토요 집중(40분, 50분)
- 주3회 : 월,수,금/화,목,토
- 주5회 : 월~금
수강권 별 횟수 일괄 등록 (예: 25분 36회권 수강 신청 시 36회 일괄 등록)
1회 수강권은 자유예약으로만 신청 가능
수강중지	
2주 이상 수강 진행 불가 시 수강 중지 가능
* 수강 중지 가능 횟수
- 1회 수강권: 중지 불가
- 4회/8회/12회/20회/24회 수강권: 1회
- 36회 수강권: 3회
수강시간 2시간 전까지 홈페이지에서 중지 가능
중지 후 2주 이내 재등록 불가 / 중지 후 6개월 이내 재등록해야 함
기한 내 미등록 시 등록된 전체 수강권 및 잔여 횟수 소멸되어 사용 불가
재등록 시 담당 강사님 변경될 수 있음
수강변경	
당일 수강 2시간 전까지 변경 가능
시간, 요일, 담당 강사, 과정 변경 가능
변경 일자 이전까지는 기존 수강 진행
진행 예정인 수강 또는 연기가 있을 경우 수강 변경 불가
수강 변경 후 취소 불가
연장등록	
수강권이 있을 경우 기존 동일 수강 자동 연장 등록
1회 수강권은 자동 연장 불가
결제 마지막 수강이 끝나기 일주일 전, 이틀 전, 종료 당일 문자 안내
수강 마지막 종료 전일까지 결제 확인되어야 동일 수강 연장 등록
연장 등록 원치 않을 경우 수강 마지막 일자까지 고객센터로 요청
일반 연기 및
보강 연기	
수강권 당 가능한 횟수 내에서 일반 연기 또는 보강 연기 가능
* 수강 연기 가능 횟수
- 1회 수강권: 연기 불가
- 4회 수강권: 총 1회 연기 또는 보강 연기 가능
- 8회 수강권: 총 1회 연기 또는 보강 연기 가능
- 12회 수강권: 총 2회 연기 또는 보강 연기 가능
- 20회 수강권: 총 3회 연기 또는 보강 연기 가능
- 24회 수강권: 총 3회 연기 또는 보강 연기 가능
- 36회 수강권: 총 6회 연기 또는 보강 연기 가능
■ 일반연기

등록된 수강의 마지막 수강 일자가 다음 수강 일자로 미루어 지는 것 (하루 전, 자정까지 신청 가능)
예시) 12회 수강 기준 8월 3일~8월 28일 경우 1회 연기 시 8월 30일 종료 일자로 변경
다중 수업의 경우 각각의 수강권에서 연기 신청해야 함
수강 하루 전일까지 가능 횟수 내에서 신청가능
■ 보강연기

등록된 수강 시간/요일에 관계 없이 변경하여 수강하는 것
등록된 수강일 포함 2일 이내로 조정하여 선택 가능
예시) 월요일 15:00 수강 -> 월요일 6:00~화요일 23:30 중 선택 가능
토요 집중 수업은 보강 연기 불가
수강 당일에만 신청 가능하며, 수강 1시간 전까지 신청 가능
일반연기/보강연기 신청 후 취소 불가
연기 신청은 PC, 모바일 홈페이지에서만 신청 가능
자유예약 수강은 연기 및 보강 연기 불가
수업 휴강일	
한국 공휴일: 휴강
필리핀 공휴일: 부활절 2일, 연말(12월 31일) 휴강
대체 공휴일의 경우 당사 사정에 따라 휴강 여부 사전 공지
자유예약	
사용기간은 자유예약 수강 등록일로부터 공휴일 포함 30일이며, 미 소진 시 잔여 횟수는 자동 소멸 됨
자유예약은 수강 연기, 보강연기, 수강 시간 변경, 수강 중지, 수강 취소 불가
홈페이지에서만 신청 가능(유선, 카카오톡 신청불가)
수강권 단위로 신청 가능
1일 횟수 제한 없이 가능한 잔여 횟수 내에서 수강신청 가능
수강 신청은 수강 시작 시간 1시간 전까지 신청 가능하며, 신청일로부터 2일 이내로 예약 가능
(예시: 8월 3일 수강 신청 시 8월 4일 수강까지 예약 가능)
자유예약 수강은 과정 변경만 가능 (고객센터 통해서 수강 2시간 전까지)
대체강사 수강	
강사 사정으로 인한 대체 강사 수강 진행 시 문자로 안내되며 진행을 원치 않을 경우 고객센터 통해 연기 가능
수강 시작 전 고객센터 전화 또는 카카오톡을 통해 연기 횟수 차감 없이 연기 신청 가능
수강권 양도	
양도인과 양수인 상호 협의가 되어 있는 상태에서 수강권 단위로 양도 가능 (구수강권: 1개월 단위)
횟수 분할 및 잔여횟수 양도 불가
수강권 사용 규정	
구매년도로부터 2년까지 사용 가능하며 미사용 시 소멸됨
구매일로부터 30일 이내 수강권 등록 권장
수강중인 경우 유효기간이 경과되어도 소멸되지 않음. (중지 시 소멸)
빅박스 수강권
사용 규정	
구매일로부터 30일 이내 등록해야 하며, 미등록 시 사용 불가함
양도 및 중지 불가
등록과 동시에 사용 시작됨
AI영문교정 이용권
사용 규정	
이용권 사용 기간은 등록일로부터 공휴일 포함 30일이며, 미 소진 시 잔여 단어 수는 자동 소멸됨
양도 및 중지 불가
등록과 동시에 사용 시작됨
녹화 영상	
수강은 자동녹화 되며 3주간 홈페이지에서 제공 됨
회원탈퇴	
고객센터 요청 또는 1:1 게시판 요청 시 탈퇴 가능하며, 탈퇴 후 재복구 불가
기존 수강 이력 및 잔여 수강권 등 모두 삭제됨
프리미엄 앤보임
수강 규정	
수강등록은 고객센터 통해 가능하며 평일 50분+토요 50분 수업이 동시에 시작되어야 함
(주6회 수업 개념)
수업 진행 시 교재 구매 필수
수강 변경: 고객센터 통해 변경 가능 (당일 변경 불가, 고객센터 수강 상담 운영시간 내)
수강 연기: 수강권 당 가능한 횟수 내에서 일반 연기 가능 (보강연기 불가, 하루 전 자정까지 신청 가능)
* 수강 연기 가능 횟수
- 평일 수업: 일반 연기 3회 가능
- 토요 수업: 일반 연기 1회 가능
수강 중지: 수강권 당 1회 가능
- 수강 시간 2시간 전까지 고객센터 통해 중지 가능
- 중지 후 2주 이내 재등록 불가 / 중지 후 6개월 이내 재등록해야 함
- 중지 시 평일, 토요 수업 모두 중지되며 재등록 시에도 두 수업 동시에 시작되어야 함
